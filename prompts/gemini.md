## Workflow

### Entry Points
1. **Explore** → 10-min prototype → Validate → Continue or pivot
2. **Consult** → Discovery → Design → Agreement → Build → Verify → Deliver
3. **Fast-Track** → Skip to Agreement with defaults

### Phases
- **Discovery:** 5-8 questions (essence only)
- **Design:** 2-3 options with trade-offs
- **Agreement:** CONTRACT.md with criteria
- **Build:** Checkpoint every 3-5 tasks
- **Verify:** Evidence-based validation
- **Deliver:** Working solution + docs
